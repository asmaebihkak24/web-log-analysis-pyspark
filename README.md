# Web Log Analysis with PySpark

This project analyzes NASA web server logs using PySpark and visualizes the results with Power BI.

## Dataset
NASA web logs (July & August)
The NASA Web Server Logs dataset used in this project is too large to be stored directly in this repository.

You can download it from:
https://ita.ee.lbl.gov/html/contrib/NASA-HTTP.html

## Technologies
- Python
- PySpark
- Power BI

## Analysis
- Data cleaning
- Handling missing values
- HTTP status code distribution
- Top HTTP methods
- Error rate analysis

## Dashboard
Interactive Power BI dashboard for web traffic analysis.
